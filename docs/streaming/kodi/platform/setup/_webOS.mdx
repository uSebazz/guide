Pre-requisites for installing Kodi on webOS (LG Smart TVs):
- Any webOS 4 or later TV (except for K5LP based TVs)
- A computer connected to the same network as your TV
- [DevMode manager for webOS](https://github.com/webosbrew/dev-manager-desktop)
- [A Kodi ipk file](https://kodi.tv/download/webos)

**Steps to install Kodi on webOS:**

1. If unsure choose Developer Mode here. Follow the steps to set up a LG developer account, install the Developer Mode app and turn it on. Alternatively, if your TV is already rooted you can set up the SSH connection instead and skip to step 3.
<img src="/img/Step1_Connection.png" alt="Step 1 Connection" width="960" height="540" />

2. Follow the steps to create a LG developer account, install the developer mode app and enable developer mode.
<img src="/img/Step2_DevMode.png" alt="Step 2 Dev Mode" width="960" height="540" />

3. Click install and select the Kodi ipk file you downloaded earlier.
<img src="/img/Step3_Install.png" alt="Step 3 Install" width="960" height="540" />

4. Wait for the installation to complete
<img src="/img/Step4_Wait.png" alt="Step 4 Kodi App" width="960" height="540" />


**Limitations**
- Binary addons are not available from the repository
